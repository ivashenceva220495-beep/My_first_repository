# System Analyst Portfolio (я только учусь)

Репозиторий с практическими кейсами, артефактами и шаблонами, которые я использую в работе системного аналитика.

## 📂 Структура

- `/requirements` — требования (SRS, user stories, use cases)
- `/diagrams` — модели в BPMN, UML, ERD (PlantUML, Mermaid)
- `/api` — OpenAPI спецификации, коллекции Postman
- `/sql` — запросы для анализа данных и витрин
- `/nfr` — SLO/SLI, расчёты нагрузки, ТЗ на интеграции
- `/templates` — чеклисты, RTM, бланки оценки задач

## 🛠️ Инструменты

- Моделирование: PlantUML, Draw.io, Miro
- API: Swagger Editor, Postman
- SQL: PostgreSQL, DBeaver
- Документация: Markdown, Notion, Confluence

## 📌 Ключевые кейсы

| Кейс | Что сделано |
|------|-------------|
| Интернет-магазин | BPMN коллаборация, OpenAPI для платежного шлюза, ERD заказов |
| Платёжный агрегатор | Sequence диаграмма с асинхронными вебхуками, словарь ошибок |
| Аналитика продаж | Витрина данных, SQL на оконные функции, дедупликация |
